# 以简洁有效的方法推动人工智能进步

[English](./README.md) | [简体中文](./README.zh-CN.md)

</div>

AIResearch-Group 是一个开放的人工智能研究团队，致力于研究和开发**简洁、有效且具有广泛适用性的人工智能方法**。

我们相信，人工智能领域真正有意义的进步，并不总是依赖越来越复杂的模型架构，也不一定需要不断叠加大量模块和技巧。相比之下，我们更希望从重要而本质的研究问题出发，通过**简洁、严谨且真正有效的解决方案**推动人工智能技术进步。

我们目前关注的研究方向包括：

- 多模态学习与表征学习
- 大语言模型与后训练
- 迁移学习与有限监督学习
- 可信与安全人工智能

我们重视具有以下特点的研究：

- **简洁**：核心思想能够被清楚理解并方便实现
- **有效**：性能提升主要来自核心机制本身
- **通用**：方法能够适用于不同模型、数据和任务
- **实用**：方案具有良好的可复现性，并便于实际应用与部署

我们希望通过严谨的学术研究和高质量的开源项目，贡献真正有价值的思想、方法与工具，共同推动人工智能技术不断向前发展。

> **一个小而有效的创新，也可以推动 AI 向前。**
